Pixel Lands - Characters Demo v1.0
------------------------------

Thank you for downloading my asset pack! This pack is a demo of the Pixel Lands - Characters asset pack and contains a sample character with a basic set of animations as well as a demo of the layered sprite sheets for creating custom characters. Check out the full version of the asset pack for more premade characters, more animations and tons more customization options!

See the instructions.txt file in the Guides folder for more information on how to use the layered sprite sheets for character customization.
