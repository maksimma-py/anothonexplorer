Thanks for downloading Pixel Lands - Characters Demo! This pack is a demo of the Pixel Lands - Charactes asset pack which can be found at https://trislin.itch.io/pixel-lands-characters

This demo contains one character that's ready to use as is (Adventurer.png). It also has layered sprite sheets you can use to create your own characters or build character customization into your game. There are only a small number of customization options available in this demo. Check out the full version for more hairstyles, skin tones and outfits!

See below for further instructions on using the layered sprite sheets.

***************************
~*~ GENERAL INFO ~*~
***************************

All animations have 4 directions (down, left, right, up). Characters are roughly 28px tall and are designed to work with environments that use a 16x16px tile size.

All sprite sheets in this pack have the same layout which is described in the guide.png file. All frames in the sprite sheets are 48x48px to make slicing easy and provide some padding for larger animations.

Animations included in this demo:

idle - 2 frames
walk - 4 frames
run - 6 frames
pickaxe - 4 frames

***************************
~*~ ANIMATION TIMINGS ~*~
***************************

Some animations will look better with different frames played at different speeds rather than all frames at the same speed. Below are some suggested frame timings (in ms) for each animation but feel free to experiment!

idle 1000/500
walk 200/200/200/200
run 133/133/133/133/133/133
pickaxe 150/250/150/150

***************************
~*~ LAYERS ~*~
***************************

To create custom characters with this pack or implement a customization system within your game you will need to use the layered sprite sheets. These can be found in folders starting with a number (eg. 3_Hair). Layered sprite sheets need to be arranged in a specific order when building a character to ensure they overlap correctly. Sprite sheets are grouped into folders denoting the order they should be placed in. The sorting order for the folders is as follows:

HIGHEST

1_Tools
2_Hats - note: there are no hats included in this demo
3_Hair
4_Tops
5_Bottoms
6_Feet
7_Base
8_Tools_Under

LOWEST

**NOTE: Tools have both a 1_Tools and 8_Tools_Under sprite sheet so make sure you use both to avoid missing tool sprites in certain frames.**
